Medicare Pharmacy Website

A complete pharmacy management system with prescription upload, medicine ordering, and doctor consultation features

View the live website embedded below. The website features:

  • Medicine ordering system
  • Prescription upload functionality
  • Doctor consultation booking
  • Responsive design for all devices
  • Secure payment integration

Key Features

Medicine Database

Comprehensive database of medicines with search, filter, and categorization features.

Digital Prescription

Upload and manage prescriptions digitally with secure cloud storage.

Doctor Consultation

Online video consultation with certified doctors and specialists.

E-commerce System

Complete shopping cart, checkout, and order tracking system.

Home Delivery

Trackable home delivery with real-time updates and notifications.

Mobile Responsive

Fully responsive design that works perfectly on all devices.

How to Run This Project Locally

  1. Clone the repository:
    git clone https://github.com/saniyathakur13/medicare.git
  2. Navigate to project directory:
    cd medicare
  3. Open in browser: Open index.html in your preferred web browser
  4. For local server (optional): Use VS Code Live Server or Python's HTTP server:
    python -m http.server 8000
  5. Access locally: Open http://localhost:8000 in your browser
